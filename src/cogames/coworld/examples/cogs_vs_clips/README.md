# Cogs vs Clips Coworld

This example packages the Cogsguard Cogs vs Clips game as a Coworld-compliant Cogame.

## Build Images

From the repo root:

```bash
docker build -f packages/cogames/src/cogames/coworld/examples/cogs_vs_clips/game/Dockerfile -t coworld-cogs-vs-clips-game:latest .
docker build -f packages/cogames/src/cogames/coworld/examples/cogs_vs_clips/player/Dockerfile -t coworld-mettagrid-policy-player:latest .
```

## Play Locally

From the repo root:

```bash
uv run cogames coworld play packages/cogames/src/cogames/coworld/examples/cogs_vs_clips/coworld_manifest.json
```

Open the printed `/global` URL to watch the live MettaGrid replay stream. The game image serves a websocket-capable
MettaScope build that is generated by the Dockerfile; the page also includes a simple canvas fallback. Open any printed
`/player?slot=...&token=...` URL to inspect a cog, take over that slot, and release it back to the policy player.

## Certify

From the repo root:

```bash
uv run cogames coworld certify packages/cogames/src/cogames/coworld/examples/cogs_vs_clips/coworld_manifest.json
```

## View A Replay

After `play` or `certify` writes a replay artifact, start a replay viewer from the repo root:

```bash
uv run cogames coworld replay packages/cogames/src/cogames/coworld/examples/cogs_vs_clips/coworld_manifest.json path/to/replay.json
```
